## CST2550 Software Engineering Management and Development 

## Project Topic

Our group was tasked with developing a **Hospital Outpatient Booking System**.

The system is designed to manage outpatient appointment records, allowing efficient storage, retrieval, and searching of patient booking data. A key requirement of the project is the use of a custom-built data structure to support these operations, alongside a detailed analysis of algorithm efficiency.

## Project Overview


This project involves the design and implementation of a Hospital Outpatient Booking System using C# (.NET). The system focuses on managing booking data efficiently through the use of custom data structures and algorithms developed entirely by the team.

As an initial stage, the team designed and prototyped the user interface using Figma. This allowed for early visualisation of the system layout, improved usability planning, and ensured consistency before development began.

Following the design phase, the frontend was implemented using HTML, CSS, and JavaScript to create a responsive and user-friendly interface. The backend system was then developed in C# (.NET), supporting core functionalities such as storing patient booking information, searching records, and interacting with a SQL database.

In addition to implementation, the project required justification of design decisions, time-complexity analysis, and thorough testing to ensure reliability and performance.

The development process follows a structured, team-based approach inspired by Agile principles, ensuring collaboration, accountability, and continuous improvement throughout the project lifecycle.
